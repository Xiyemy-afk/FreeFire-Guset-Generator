# FF Guest Account Generator Termux

Tool sederhana untuk reset / generate akun Guest Free Fire menggunakan Termux.

## Fitur
- Auto detect folder Free Fire (Global, ID, MAX, dll)
- Backup & Restore otomatis
- Mudah digunakan
- Support ZArchiver method

## Cara Install
```bash
pkg install git -y
git clone https://github.com/Xiyemy/FF-Guest-Account-Generator-Termux.git
cd FF-Guest-Account-Generator-Termux
chmod +x ff-guest-reset.sh
./ff-guest-reset.sh
